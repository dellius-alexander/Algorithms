Algorithms

An algorithm is a procedure to accomplish a specific task. An algorithm must solve a general, well-specified problem. An algorithmic problem is specified by describing the complete set of instances it must work on and of its output after running on one of these instances. This distinction, between a problem and an instance of a problem, is fundamental. An algorithm is a procedure that takes any of the possible input instances and transforms it to the desired output.
